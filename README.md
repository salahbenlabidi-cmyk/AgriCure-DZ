# AgriCure-DZ
منصة ذكية مدعومة بالذكاء الاصطناعي (AI) لتشخيص أمراض المحاصيل الزراعية في الجزائر، تهدف لتعزيز الأمن الغذائي وتقليل استخدام المبيدات العشوائية." أو بالإنجليزية (وهو المفضل في GitHub): "AI-powered platform for crop disease diagnosis in Algeria, aimed at enhancing food security and optimizing pesticide use." اللمسة
